# 회원가입

API Path: /api/v1/users/signup
선택: POST

## **Request Header**

## Request Body

```sql
{
  "name": "홍길동",
  "email": "test@example.com",
  "password": "1234abcd"
}
```

## Response (성공)

```sql
{
  "success": true,
  "code": 200,
  "message": "회원가입 성공",
  "data": {
    "id": 1,
    "name": "홍길동",
    "email": "test@example.com",
    "createdAt": "2025-10-02T12:00:00Z"
  }
}
```

## Response (실패)

```sql
{
  "success": false,
  "code": 400,
  "message": "이미 존재하는 이메일입니다."
}
```
