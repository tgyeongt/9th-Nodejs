# 로그인

API Path: /api/v1/users/login
선택: POST

## **Request Header**

## Request Body

```sql
{
  "email": "test@example.com",
  "password": "1234abcd"
}
```

## Response (성공)

```sql
{
  "success": true,
  "code": 200,
  "message": "로그인 성공",
  "data": {
    "token": "토큰 자리"
  }
}
```

## Response (실패)

```sql
{
  "success": false,
  "code": 401,
  "message": "이메일 또는 비밀번호가 올바르지 않습니다."
}
```
