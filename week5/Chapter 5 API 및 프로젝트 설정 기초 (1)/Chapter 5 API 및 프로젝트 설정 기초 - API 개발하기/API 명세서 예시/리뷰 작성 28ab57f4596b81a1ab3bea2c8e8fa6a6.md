# 리뷰 작성

API Path: /api/v1/store/reviews
선택: POST

## **Request Header**

```sql
Authorization: Bearer <token>
Content-Type : application/json
```

## Request Body

```sql
{
  "missionId": 101,
  "content": "좋아요!",
  "rating": 5
}
```

## Response (성공)

```sql
{
  "success": true,
  "code": 201,
  "message": "리뷰 작성 성공",
  "data": {
    "id": 1001,
    "missionId": 101,
    "content": "좋아요!",
    "rating": 5,
    "createdAt": "2025-10-02T12:00:00Z"
  }
}
```

reviews는 가게에 종속되는 것이 맞다

마이페이지에서 reviews를 조회하고 싶다면 api를 따로 만든다.
