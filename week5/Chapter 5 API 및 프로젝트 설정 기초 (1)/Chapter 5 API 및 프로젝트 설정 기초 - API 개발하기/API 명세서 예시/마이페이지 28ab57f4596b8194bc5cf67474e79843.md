# 마이페이지

API Path: /api/v1/users/my
선택: GET

## **Request Header**

```sql
Authorization: Bearer <token>
Content-Type : application/json
```

```sql
{
	"success": true,
	"code": "200",
	"message": "요청이 정상적으로 처리되었습니다.",
	"data": {
		 "nickname": "홍길동",
		 "email": "test@example.com",
		 "point": 2500
		 "phone": "010-1234-5678"
	}
}
```
